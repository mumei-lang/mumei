Mumei (無銘)

Mathematical Proof-Driven Programming Language — verified by Z3, compiled to LLVM IR, transpiled to Rust/Go/TypeScript.

GitHub

Install

curl -fsSL https://mumei-lang.github.io/mumei/install.sh | bash

or

brew install mumei-lang/mumei/mumei

No Rust toolchain required. Detects OS/arch automatically.

Quick Start

1Create a project
mumei init my_app && cd my_app
2Build (verify + codegen + transpile)
mumei build src/main.mm -o dist/output
3Check your environment
mumei inspect
